# Coffee_Machine_OOP (Python)

A Python console-based coffee machine simulation built using Object-Oriented Programming principles. The program models real-world components such as a menu, coffee maker, and payment system using separate classes, handling user input, resource management, and transactions in a structured and scalable way.

# ▶️ How to Run

1. Make sure you have Python installed on your computer.

2. Download or clone this repository.

3. Ensure all files are in the same folder:

   main.py
   menu.py
   coffee_maker.py
   money_machine.py

4. Open a terminal (or your preferred Python IDE).

5. Navigate to the folder containing the files.

6. Run the script using:

   python main.py

7. Follow the on-screen prompts to use the coffee machine.


# 🧠 Concepts Practiced

- Object-Oriented Programming (OOP)
- Creating and using classes and objects
- Interacting with multiple modules
- Encapsulation of logic within classes
- Using methods to manage state and behavior
- Avoiding duplication (single source of truth)
- Using intermediate variables for cleaner code
- Controlling program flow using boolean return values
- Combining conditions to manage execution flow
- Separating responsibilities across components (menu, machine, payment)
- Managing resources and state within objects
- Writing and using docstrings for documentation
